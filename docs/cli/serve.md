# Serve Command

`burger-api serve` starts the development server with hot reload (restarts on file changes).

**Options:**

- `-p, --port <port>` — Port (default: `4000`)
- `-f, --file <file>` — Entry file (default: `src/index.ts`)

**Example:**

```bash
burger-api serve --port 4000
```

See [CLI Tool](/docs/getting-started/cli).
